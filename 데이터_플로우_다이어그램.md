## 8. 게임 미션 완료 프로세스

```mermaid
sequenceDiagram
    participant U as 사용자
    participant GP as 게임 상세 페이지
    participant LS as localStorage
    participant DM as 일일 미션 페이지
    participant UI as UI 컴포넌트
    
    U->>GP: 미션 버튼 클릭
    GP->>LS: gameStates[index] 업데이트
    GP->>UI: 미션 상태 렌더링
    
    Note over GP: 3개 미션 완료 시
    
    U->>GP: "2 USDT 받기" 클릭
    GP->>LS: gameStates[index] = 4
    GP->>LS: usdt += 2
    GP->>LS: history.push(거래내역)
    GP->>UI: 리워드 오버레이 표시
    GP->>UI: LINE 알림 표시
    
    Note over DM: 다른 페이지에서 확인
    
    DM->>LS: 상태 조회
    DM->>UI: 진행률 업데이트
    
    Note over DM: 3게임 완료 시 뽑기권 지급
    
    DM->>LS: tickets += 3
    DM->>UI: 뽑기권 리워드 표시
```
